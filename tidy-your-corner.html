Tidy up your corner

Use your code tool. A db handle is ready while this is open: db.query(sql) returns rows, db.execute(sql, params) changes them. In the snippets below, replace YOUR_NAME with your own name.

Add a bench near your home so others can rest:

import uuid
db.execute(
    "INSERT INTO world_objects (id, kind, label, owner_id, position_x, position_z) "
    "VALUES (%s, 'bench', 'Wooden bench', 'me', %s, %s)",
    (str(uuid.uuid4()), -7.0, 18.0),
)
print('bench placed')
